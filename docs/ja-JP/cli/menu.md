# メインメニュー

`npx zcf` で表示されるインタラクティブメニュー。全機能を番号でたどれるため、パラメータを覚える必要がありません。

## 主な項目

- **1 完整初期化**：`zcf init` 相当（API/モデル/MCP/ワークフロー/出力スタイル）
- **2 ワークフロー导入**：必要なワークフローのみ追加
- **3 API/CCR 設定**：公式ログイン/API Key/CCR を選択
- **4 MCP 設定**：Context7 / Open Web Search などを選択導入
- **5 デフォルトモデル**：モデル/Fallback を設定
- **6 AI メモリ & 出力スタイル**：`CLAUDE.md` 編集とスタイル切替
- **7 環境権限**：推奨環境変数/権限を確認・適用
- **R CCR 管理**：CCR の起動/停止/ログ
- **U ccusage**：利用状況の確認
- **L CCometixLine**：ステータスバーの導入/更新
- **+ 更新チェック**：Claude Code/Codex のアップデート確認

## ヒント

- 初回はメニューで各項目を一度通しておくと流れを把握しやすい  
- コマンドを覚えた後でも、更新や確認作業に活用するとミスが減ります  
- Codex モードで開くには `npx zcf --code-type codex` を指定してください
